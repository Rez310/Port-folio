Mes Compétences Techniques

Compétences acquises dans le cadre du BTS SIO option SISR, organisées selon le référentiel officiel

Bloc 1 : Gérer le Patrimoine Informatique

Recenser et identifier les ressources numériques

Mise en place d'un système d'inventaire automatisé avec GLPI permettant de tracer tous les équipements informatiques de l'organisation.

  • Déploiement et configuration de GLPI sur serveur Linux
  • Installation d'agents d'inventaire sur les postes clients
  • Création de rapports automatisés d'inventaire matériel et logiciel
  • Gestion du cycle de vie des équipements (acquisition, affectation, maintenance, fin de vie)

Exploiter les référentiels et normes

Application des bonnes pratiques ITIL pour la gestion des services informatiques.

  • Mise en œuvre des processus ITIL (gestion des incidents, des changements)
  • Respect des normes de sécurité (ISO 27001)
  • Documentation technique standardisée

Vérifier les niveaux d'habilitation

Configuration des droits d'accès et des politiques de sécurité selon le principe du moindre privilège.

  • Gestion des groupes et utilisateurs dans Active Directory
  • Configuration des ACL sur les ressources réseau et systèmes
  • Mise en place de l'authentification multi-facteurs

Gérer des sauvegardes

Mise en place de stratégies de sauvegarde pour assurer la continuité de service.

  • Configuration de sauvegardes automatiques (complètes, différentielles, incrémentielles)
  • Tests de restauration réguliers
  • Mise en place du plan de reprise d'activité (PRA)
GLPI Active Directory ITIL Sauvegardes ISO 27001

Bloc 2 : Répondre aux Incidents et Demandes d'Assistance

Collecter, suivre et orienter les demandes

Gestion centralisée des demandes utilisateurs via un système de ticketing GLPI.

  • Configuration du système de gestion de tickets sur GLPI
  • Catégorisation et priorisation des incidents selon leur impact
  • Routing automatique vers les équipes compétentes
  • Suivi des SLA (Service Level Agreement)

Traiter les demandes services réseau et système

Résolution d'incidents sur l'infrastructure réseau et les serveurs.

  • Diagnostic et résolution de problèmes de connectivité réseau
  • Dépannage des services Windows Server (AD, DNS, DHCP)
  • Intervention sur les systèmes Linux (Apache, MySQL, SSH)
  • Configuration de VLANs et de routage

Traiter les demandes concernant les applications

Support applicatif et assistance aux utilisateurs sur les logiciels métiers.

  • Installation et configuration de logiciels
  • Résolution de problèmes applicatifs
  • Formation des utilisateurs finaux
  • Création de documentation utilisateur
Support N1/N2 GLPI Ticketing Windows Server Linux Réseau

Bloc 3 : Développer la Présence en Ligne

Développement et maintenance de sites web exploitant les données de l'organisation.

  • Tests de compatibilité multi-navigateurs
HTML5 CSS3 JavaScript Google Analytics SEO RGPD
